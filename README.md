# Gemini-Apps-Script-Web-App 🗣️
A project started by me (TMAPRICOT) to get access to Google Gemini 3 Flash at school using Google Apps Script. The project uses an HTML user interface front-end and a JavaScript back-end in order to communicate with the AI via an API Key.

NOTE: I used OpenAi's Codex agent to assist me in creating this, as I'm not completely familiarized with JS and I only know basic HTML. However, everything within this README has been written by me personally.

## Instructions!📜
### Step 1:
Copy the code from "Code.js" within this repo.
<img width="1283" height="816" alt="image" src="https://github.com/user-attachments/assets/1b6f8062-4d5e-481f-abd3-29544334d4e5" />
### Step 2:
In your browser (like Chrome) type "script.new" in the search field. It should open up to a tab that looks like this.
<img width="1915" height="918" alt="image" src="https://github.com/user-attachments/assets/aed734ad-e407-4e37-a18e-f9045695a169" />
### Step 3:
Rename the file as you wish. Delete everything within the file, and replace it with the code you copied earlier from here.
<img width="1187" height="943" alt="image" src="https://github.com/user-attachments/assets/85409cb3-b81f-40d6-9eee-3ca7bbd448d3" />
### Step 4:
Near the top-left corner, select the "+" icon and select "HTML" in the dropdown menu.
<img width="830" height="351" alt="image" src="https://github.com/user-attachments/assets/28cd9edf-f183-4078-80cb-298c75094896" />
### Step 5:
Come back to this repository, and copy the code from "Index.html" to your clipboard.
<img width="941" height="359" alt="image" src="https://github.com/user-attachments/assets/696ac100-96bf-4006-a558-05fbb064bfaa" />
### Step 6:
Rename the HTML file you created in the Apps Script to "Index.html" just like the name of the file containing the code you copied. Then, delete everything within the file and paste the code you got from this repo.
<img width="1560" height="943" alt="image" src="https://github.com/user-attachments/assets/919ddf3f-e93e-4cc3-b519-9a463e3a62e4" />
### Step 7:
Go back to Code.js (the one in your script) and replace "PASTE_YOUR_GEMINI_API_KEY_HERE" with your Gemini API Key. To get an API Key, follow this [tutorial](https://www.youtube.com/shorts/Tvl0hlj_aG4). Remember to click Ctrl+S to Save.
<img width="1184" height="501" alt="image" src="https://github.com/user-attachments/assets/4cd79243-e52d-4a11-a032-eb1e6254c645" />
### Step 8:
Select "Deploy" -> "New Deployment -> Gear Icon -> "Web App" -> Manage access as you wish -> Deploy -> Copy Web App URL.
<img width="756" height="597" alt="image" src="https://github.com/user-attachments/assets/83728db8-298f-4caf-bcda-f9f783054702" />

## And you're done!🥳
